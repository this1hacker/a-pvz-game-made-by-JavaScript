# 🌻 植物大战僵尸：纯手工“肝”制版 (JS Edition)

> **警告**：本项目由 HTML、CSS 和原生 JavaScript 手搓而成，不含任何框架防腐剂。食用前请确保你的浏览器没有因为过于复古而窒息。

## 📖 项目简介

这是一个基于经典游戏《植物大战僵尸》复刻的网页版小游戏。没有 Unity，没有 Unreal，只有最纯粹的 `document.createElement` 和对 `setInterval` 的爱（恨）。

在这个版本中，你将体验到：
- **极致的网格吸附**：强迫症福音，植物永远站在它该站的地方（除非你手抖）。
- **阳光经济学**：从 50 阳光起步，体验从“贫困线”到“向日葵大亨”的辛酸历程。
- **僵尸社会学**：普通僵尸、路障僵尸、铁桶僵尸，以及它们那令人头秃的行走动画。
- **元素反应系统**：寒冰减速 vs 火焰解冻，策略搭配才是王道！
- **铲子哲学**：种错了？没关系，铲子教你做人（或者教植物重新投胎）。

## 🎮 如何开始游戏

1. **打开文件**：双击 [[04-pvz.html](file://e:\project\实训项目3\a-pvz-game-made-by-JavaScript\04-pvz.html)](./04-pvz.html)。是的，就这么简单，不需要 `npm install`，不需要配置 Webpack，只需要一个能跑 JS 的浏览器。
2. **收集阳光**：点击天上掉下来的阳光，或者指望你的向日葵勤快一点。
3. **种植物**：点击上方的卡片，然后在草坪上找个空地点击左键。
   - *注意*：如果卡片是灰色的，要么是你穷（阳光不足），要么是它在冷却（CD中）。
4. **铲除错误**：点击右下角的铲子图标，激活后点击种错的植物。小心别把脑子铲没了。
5. **右键取消**：如果你选了植物但不想种了，右键点击即可取消预览。

## 🧟 僵尸档案

- **普通僵尸**：脑子不多，走得慢。
- **路障僵尸**：头上顶个路障，稍微耐打一点。
- **铁桶僵尸**：头上顶个铁桶，非常耐打。建议用土豆雷或樱桃炸弹伺候（哦等等，樱桃炸弹不能用...那多用几个土豆雷吧，或者试试**火炬树桩+双发射手**的组合拳！）。

## 🛠️ 技术栈 (也就是全是原生)

- **HTML5**: 负责搭建舞台。
- **CSS3**: 负责让界面看起来不像 90 年代的网页（尽量）。
- **JavaScript (ES5-ish)**: 负责所有的逻辑、动画、碰撞检测。没有 Canvas，全靠 DOM 操作，性能优化？不存在的，能跑就行！

## ⚠️ 已知问题 (特性)

1. **卡顿？**：如果僵尸太多，浏览器可能会喘不过气。这是正常的，毕竟我们在用 DOM 节点做游戏引擎。
2. **阳光飞行动画**：阳光飞向计数器时可能会穿模，请脑补它被吸进去了。
3. **铲子手感**：铲子激活后跟随鼠标，有时候可能会有点飘，请多练习肌肉记忆。
4. **土豆雷判定**：如果僵尸贴脸了土豆雷还没炸，可能是因为它还在“热身”。请尊重土豆的准备时间。
5. **未实装植物**：如上所述，部分植物卡片仅为视觉装饰，点击无效。请不要疯狂点击它们，卡片不会生气，但你的鼠标会。
